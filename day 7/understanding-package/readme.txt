Module is a javascript file which contain functions
  - we export functions by syntax 
  - module.exports{function-name}
  - to import module we use require method and pass our module file

A package is a group of module
 - for example react is a package
 -useState, useEffect are module of react packages

npm- Node Package Manager
- It is a default package manager of Node.js, used to install javascript libraries and tools.

React is javascript library that is used to build up user interfaces, especially for single-page applications(SPAs)
