Hexadecimal

Hexadecimal Notation: CSS allows developers to define colors using various notations, and one of the most widely used is hexadecimal notation. Hex codes consist of six characters, ranging from 0 to 9 and A to F, representing the RGB (Red, Green, Blue) values. For example, #FF5733 represents a shade of orange.
